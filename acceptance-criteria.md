# Acceptance Criteria

## Order Lookup
- User can enter order number  
- System returns matching order  
- Error shown if invalid  

---

## View Order Status
- System displays current order status  
- Status reflects latest backend update  
- Error shown if unavailable  

---

## Display Update Timing
- System displays update frequency message  
- Message is visible and clear  

---

## Contact Support
- User can access support option  
- User can submit request  
- Confirmation message is shown
