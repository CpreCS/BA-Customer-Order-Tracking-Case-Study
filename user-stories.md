# User Stories

## Story 1: Order Lookup
As a customer, I want to enter my order number so that I can view my order status.

---

## Story 2: View Order Status
As a customer, I want to view my order status so that I understand where my order is in the process.

---

## Story 3: Display Update Timing
As a customer, I want to know how often my order updates so that I understand when to expect changes.

---

## Story 4: Contact Support
As a customer, I want to contact support if needed so that I can resolve issues related to my order.
